1 вариант:
подготовка Экспортного сценария с Загрузкой констант
+ Экспортного сценария с Загрузкой ВСЕХ необходимых справочников (Подготовка и загрузка данных - с отбором по конкретному заказу!)

Возможные сложности: 
1.1. узкая выборка тестовых данных (данные конкретного заказа из отбора);
1.2. соответственно нет возможности последующего тестирования функционала, например: Покупатель с одним типом цен, один склад, ограниченаня конкретным заказом номенклатура
1.3. сложности рефакторинга: все тестовые данные в одном фича файле

2 вариант ПРЕДПОЧТИТЕЛЬНЫЙ:
выделение всех необходимых справочников
подготовка отдельных экспортных файлов по каждому справочнику (при необходимости с применением отборов)
подготовка сценария проверки заказа: с использованием экспортных сценариев загрузки констант, справочников

Сложности: первоначальные трудозатраты выше относительно первого варианта
Приемущества: удобства последующего обслуживания тестов, возможность использования экспортных сценариев в других сценариях.
